# Customer Behavior Analysis

## Overview

This project analyzes customer purchasing behavior using Python, MySQL, and Tableau. The objective is to clean and explore the dataset, extract business insights through SQL queries, and present the findings in an interactive dashboard and presentation.

The project follows a complete data analytics workflow, from data preparation to visualization and reporting.

---

## Dataset

The dataset contains customer shopping behavior, including:

* Customer demographics
* Purchase history
* Product categories
* Payment methods
* Subscription status
* Discounts and promotions
* Customer review ratings
* Seasonal purchasing trends

---

## Tools & Technologies

* **Python** – Data loading, cleaning, and exploratory data analysis (EDA)
* **Pandas** – Data manipulation
* **NumPy** – Numerical operations
* **Matplotlib & Seaborn** – Data visualization during EDA
* **MySQL** – Data storage and SQL analysis
* **SQLAlchemy & PyMySQL** – Database connection
* **Tableau** – Interactive dashboard development
* **Gamma AI** – Presentation creation

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Connected Python to a MySQL database.
* Loaded the cleaned dataset into MySQL.

### 2. Data Cleaning

* Handled missing values.
* Removed duplicates.
* Corrected inconsistent values.
* Standardized data types and formats.

### 3. Exploratory Data Analysis (EDA)

* Examined customer demographics.
* Analyzed purchasing behavior.
* Identified trends and patterns.
* Explored relationships between customer attributes and sales.

### 4. SQL Analysis

Performed SQL queries to answer key business questions, including:

* Revenue analysis
* Customer segmentation
* Product category performance
* Subscription analysis
* Seasonal sales trends
* Purchase frequency analysis
* Payment method distribution

### 5. Dashboard Development

Created an interactive Tableau dashboard featuring:

* Sales overview
* Customer segmentation
* Product performance
* Subscription insights
* Purchase trends
* Interactive filters for data exploration

### 6. Reporting

* Summarized key findings.
* Documented business insights.
* Presented recommendations based on the analysis.

### 7. Presentation

Created a professional project presentation using Gamma AI to communicate the analysis process, findings, and recommendations.

## Key Results

Some of the insights obtained include:
